# Политика конфиденциальности

**Расширение:** TabTopic Organizer
**Версия:** Beta 0.1
**Последнее обновление:** 5 июня 2026 г.

---

## 1. Сбор данных

Расширение **не собирает, не хранит и не передаёт** никакие данные третьим сторонам.

## 2. Как работает расширение

TabTopic Organizer работает **исключительно локально** в вашем браузере. Расширение:

- Читает заголовки (title) и URL-адреса открытых вкладок **только текущего окна**.
- Анализирует их на наличие ключевых слов для определения тематики.
- Использует нативный API браузера для создания визуальных групп вкладок.

Все операции выполняются на вашем устройстве. Никакие данные не покидают браузер.

## 3. Сетевые запросы

Расширение **не отправляет** никаких сетевых запросов. Доступ к интернету не требуется для работы функций группировки.

## 4. Хранение данных

Расширение **не использует** локальное хранилище (localStorage, storage API) для сохранения информации о вкладках, историях посещений или пользовательских данных.

## 5. Доступ к вкладкам

Расширение запрашивает разрешение `tabs` для:

- Получения списка вкладок в текущем окне (`browser.tabs.query`).
- Группировки вкладок (`browser.tabs.group`).

Данные о вкладках используются **только во время выполнения операции группировки** и не сохраняются после неё.

## 6. Буфер обмена и ввод пользователя

Расширение **не взаимодействует** с буфером обмена и не запрашивает ввод данных пользователя, кроме нажатий кнопок в интерфейсе.

## 7. Поставщики третьих сторон

Расширение **не интегрируется** с какими-либо сторонними сервисами, API или SDK.

## 8. Детали для детей

Расширение не предназначено для детей младше 13 лет и не собирает данные о несовершеннолетних.

## 9. Изменения в политике

В случае изменений в политике конфиденциальности обновлённая версия будет опубликована в репозитории проекта с указанием даты обновления.

## 10. Контакты

По вопросам конфиденциальности создайте issue в репозитории проекта.
